# ゲームについて
__操作方法__

タイトル画面
* エンターキー：ゲーム開始
* スペースキー：クレジット表記

メインゲーム画面
* A/Dキー,左矢印/右矢印キー：移動
* 移動中に降ってくる障害物に擦れる：擦り状態
* Rキー：リトライ
* ESCキー：終了
